## 2. 核心診断（2026-05-30・前回指示の撤回を含む）

### ⚠️ [前回指示の撤回] FAQ追加は主因ではなかった

- 時短健康 = **全97記事にFAQ設置済み（100%）** → それでもオーガニック流入 月11セッション
- やまちゃん日記 = FAQ設置は **31/149記事（約21%）** → オーガニック流入 月242セッション

**FAQが5分の1しかないやまちゃんが、22倍のオーガニック流入を得ている。**
→ FAQ構造化データは「やって損はない衛生要因」だが、流入差を生む要因では**ない**。
→ DEV-INSTRUCTIONS-20260526.md の「FAQ欠落が主因」という前提は誤り。FAQ作業は完了済みとして扱い、以下の本当の真因に注力する。

### [CRITICAL-1] 時短健康には「本物の検索ユーザー」がほぼ来ていない

- オーガニック検索 **わずか10%**（11/108セッション）。残りは Direct 66% + Referral 22%。
- デスクトップ **79%**・モバイルわずか20%。
  → 食宅配・健康ジャンルの実消費者は**圧倒的にモバイル**。デスクトップ79%は「実消費者ではない」サイン
  （= オーナーの確認アクセス・SNS/内部リファラル・開発時アクセスが大半）。
- やまちゃんは逆にモバイル56%・オーガニック73% = **外部の実検索ユーザーが主役**。

→ 時短健康の「アクセス」は実需ではなく身内。**外部検索ユーザーへの到達がほぼゼロ**が問題の本体。

### [CRITICAL-2] Googleが記事を「表示すらしていない」（インプレッション枯渇）

| サイト | 代表記事 | インプレッション | 順位 |
|---|---|---|---|
| やまちゃん | クーラーボックス比較 | **391** | 6.8位 |
| やまちゃん | バイク釣り竿積載 | **255** | 7.4位 |
| 時短健康 | ミールキット3社比較 | 45 | **42.8位（5ページ目）** |
| 時短健康 | ノッシュvs三ツ星vsマッスルデリ | 15 | **44.2位** |
| 時短健康 | マッスルデリレビュー | 4 | 2.5位（※表示数が少なすぎ） |

→ クリック以前に**インプレッションが出ていない**。順位40〜50位（検索結果5ページ目）では誰の目にも触れない。

### [CRITICAL-3] 真因は「キーワード選定がレッドオーシャン正面突破」

やまちゃんが4〜9位に入れている理由は、**大手が手薄なミッドテール複合キーワード**を取っているから：
- 勝ちクエリ:「シマノ ダイワ クーラーボックス 比較」「バイク 釣り竿 積載」「セロー225 後悔」
- これらは「釣り×キャンプ×バイク」の交差点。大手アフィリ・大手メディアが存在しない隙間。
  → 新興ドメイン（低DA）でも上位に入れる。

一方、時短健康が狙っているのは**食宅配アフィリの大激戦区**：
- 「ヨシケイ オイシックス パルシステム」→ 52位
- 「nosh マッスルデリ 比較」→ 52位
- 「マッスルデリ ナッシュ」→ 48位
  → 食宅配比較は大手アフィリサイトがDA・被リンクで支配。新興ドメインが正面から挑んでも**構造的に勝てない**。

### [補足・交絡要因] 2026年5月Googleコアアップデート（I-109）

- 自社の つりナビ-75% / 医療ブログ-55% / クリニックHP-78% が2026-05-01境界で同時急減。
- 時短健康の不振の一部はコアアップデートの影響もありうる。**すべてをオンページのせいにしない。**
- ただしやまちゃんはニッチゆえ影響が小さく、**ニッチ複合キーワード戦略はコアアップデート耐性が高い**ことの傍証でもある。

## 3. 移植すべき「やまちゃんの本質」3つ

1. **大手が手薄なミッドテール複合キーワードを取る**（最重要レバー）
   - 「商品名 単体」「2社比較」のレッドオーシャンをやめる。
   - 時短健康の固有武器=**精神科医の権威**を活かせる「状況×感情×食」の複合クエリへ寄せる。
2. **記事を検索の入口にする**（トップ集中をやめる）
   - やまちゃんは記事77%・トップ23%。時短健康はトップ71%。
   - 記事が個別に順位を持てば、それぞれが流入の入口になる。
3. **比較記事に実数値・実体験を入れて滞在を伸ばす**
   - やまちゃんのレビュー記事はCTR14.8%・滞在24分の記事もある。E-E-A-Tの核。

## 4. 実行タスク（優先順・データ駆動）

### Task 1【最優先・最大レバー】キーワード戦略の転換 — 「精神科医×状況」複合クエリへ

**狙い**: 食宅配大手が絶対に書けない、精神科医の権威が活きる複合ロングテールを20本選定し、新規記事 or 既存リライトで取りに行く。

**狙うクエリの方向性（大手が手薄・E-E-A-Tで勝てる）**:

| ❌ やめるクエリ（レッドオーシャン） | ✅ 狙うクエリ（精神科医が勝てる隙間） |
|---|---|
| nosh マッスルデリ 比較 | うつ 自炊できない 食事 |
| ヨシケイ オイシックス 比較 | 適応障害 食事 作れない |
| 食宅配 比較 おすすめ | 夜勤明け 食事 作りたくない |
| マッスルデリ 口コミ | 一人暮らし 自炊 続かない しんどい |
| 冷凍弁当 ランキング | 産後 ごはん 作れない 限界 |
| | 仕事 疲れて 何も食べたくない |
| | ストレス 食べ過ぎ 夜 やめたい |
| | 食欲ない 痩せた 疲れ 原因 |

**ルール**:
- これらの複合クエリで記事を書き、本文中で自然に食宅配ProductCardへ接続する（クエリ=入口、商品=出口）。
- 必ず `targetKeyword:` frontmatter に狙うクエリを明記。
- タイトル先頭にクエリのコア語を入れる（例:「夜勤明けに食事を作りたくない人へ — 精神科医が勧める"作らない"選択肢」）。
- 精神科医の一人称コメント（先生）を必ず1箇所以上。これが大手アフィリとの決定的差別化。
- **薬機法**: 「治る」「改善する」は禁止 →「ラクになることがある」「負担が減る」。`node validate-yakkiho.js` で公開前チェック。

**進め方**: まず20クエリを確定 → 既存97記事で近いものはリライトでクエリを寄せる / 該当なしは新規5本を先に書く。
