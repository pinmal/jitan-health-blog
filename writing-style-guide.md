# 時短×健康ブログ 文章術ガイド
> 6つの人気ブログ徹底分析から抽出した「人間らしく・自然に商品紹介できる」書き方の法則

作成日: 2026-04-26
参考ブログ: kochikablog.com / takushoku-repo.com / blog-funl-ife-29.com / sakurameblog / bizspa.jp / note村上はな

---

## 一言で言うと

> **「先に友達になってから、紹介する」**
> 商品を紹介するのは「問題→失敗→発見→正直評価」の4段階を踏んだ後。
> 順番を守るだけで、売り込み臭がゼロになる。

---

## 法則1：冒頭は「読者の心の声」から始める

❌ ダメな例（商品から始める）
```
今回はナッシュを紹介します。忙しい方に人気の宅食サービスです。
```

✅ 良い例（痛みから始める）
```
「今日も帰りが21時か…。また惣菜だ。」

共働き5年目にして、この罪悪感にいいかげん疲れた。
```

✅ 別パターン（矛盾・疑問から始める）
```
「GREEN SPOON、高くない?」

正直、最初はそう思った。でも1ヶ月使い続けて考えが変わった話をする。
```

✅ 別パターン（転機から始める）
```
夫婦で共働きになった3月、最初に困ったのは「夕飯を誰が作るか」じゃなかった。
「何を食べるか考えること自体がしんどい」というのが本当の問題だった。
```

**ポイント**
- 最初の1〜2文は14文字以内
- 読者が「あ、わかる」と思える状況描写
- 商品名は冒頭100文字に出てこなくていい

## 法則2：自分の弱さ・怠惰を正直に書く

人気ブロガーが共通してやっていること：**「私は完璧じゃない」を先に言う**

```
正直に言うと、料理が嫌いです。
「何を食べるか考えるのが面倒くさい」という横着な理由でこのサービスを始めました。
```

```
最初はおいしくなかった。本当に。
でも3ヶ月続けたら感想が変わったので、それをそのまま書く。
```

```
「栄養バランス気にしてます」みたいな意識高い感じは全くなくて、
単純に「考えたくない」「決めたくない」が出発点でした。
```

**なぜ効く？**
→ 読者は「ちゃんとした人の成功体験」より「自分と同じ雑な人の話」に共感する。
→ 弱みを最初に見せると「この人、正直だな」という信頼が生まれる。

## 法則3：数字は「継続期間」「試した数」「具体的な金額」

数字なしの文章は信頼されない。でも数字のつけ方が重要。

❌ 弱い数字
```
しばらく使いましたが、栄養バランスが良いと感じました。
```

✅ 強い数字
```
週3日、4ヶ月使い続けた結論を言う。（累計50食以上）
```

```
昼食1食あたり464円。外食なら余裕で800〜1,000円するから、
逆に安い……というのが使い始めて気づいたことだった。
```

```
試したのはナッシュ・GREEN SPOON・ヨシケイ・三ツ星ファーム・BASE FOOD の5つ。
それぞれの「続けられる人・続けられない人」をはっきり書く。
```

## 法則4：弱点を書く（これが一番大事）

**弱点を隠すブログ → ステマっぽい → 信頼されない → 買わない**
**弱点を書くブログ → 正直そう → 信頼される → 買う**

弱点の書き方パターン：

```
（弱点）価格は正直、安くない。1食あたり700〜900円する。
（文脈）でも、外食と比較すると別に高くないし、
       食材費・光熱費・時間コストを足したら自炊よりトントンかもしれない。
```

```
（弱点）スムージーは、子どもには評判が悪かった。「野菜の味がする〜」と言われた。
（文脈）これは完全に大人向けのサービスで、子どもに飲ませるものじゃない。
       そういう意味では、想定ターゲット通りなのかも。
```

```
（弱点）当たり外れがある。ナッシュは特に、副菜が似たりよったりだと感じた。
（文脈）解決策は「2サービス併用」。週3ナッシュ、週2GREEN SPOONにしたら飽きなくなった。
```

弱点は「問題提起→文脈説明→解決策」の3点セットで書く。

## 法則5：商品紹介の黄金フロー

```
[1] 読者の痛み（問題の描写）
↓
[2] 自分の失敗・迷走（信頼構築）
↓
[3] このサービスとの出会い（発見ストーリー）
↓
[4] 正直な体験談（良い点＋弱点）
↓
[5] 「こんな人に向いてる・向いてない」（ターゲット明示）
↓
[6] キャンペーン情報 + リンク（最後の後押し）
```

フロー例：
```
[1] 帰宅後19時、何も考えたくないのにご飯の問題が待ってる。

[2] ヨシケイを試した。でも週5日指定配達が思ったより窮屈だった。
   次にナッシュ。これは続いたけど、3週間で飽きた。

[3] 職場の先輩に「GREEN SPOONは野菜多くて好き」と言われて試してみた。

[4] 正直な感想↓（ここから詳細）

[5] こんな人に合う：健康意識あり・野菜不足が気になる・40〜50代女性
   合わない人：とにかく安く済ませたい・量を食べたい・甘めの味が好き

[6] 今なら初回30%オフ。迷ってる人は↓（リンク）
```

## 法則6：「案件ではありません」宣言と、その使い方

note村上はなが「案件ではありません」と書いて信頼を得た手法。

当ブログの場合：
```
※ このブログはアフィリエイト広告を含みます。
  ただし「実際に使って良かったもの」しか紹介しません。
  使ってみてダメだったものも、ダメだったと書きます。
```

さらに強力なのが：
```
正直に言うと、このサービスは苦手な人もいると思う。（←弱点を先に出す）
それでも私が紹介するのは、刺さる人には本当に刺さると思うから。
```

## 法則7：見出しは「読者の疑問そのまま」

❌ ダメな見出し（説明的すぎる）
```
GREEN SPOONの特徴と価格について
```

✅ 良い見出し（読者の頭の中にある言葉）
```
GREEN SPOONって高くない？ 実際の月額を計算してみた
```
```
まずいって本当？ 30食食べた正直な感想
```
```
忙しい共働きに向いてるの？ 1ヶ月使って気づいた向き不向き
```
```
ナッシュとGREEN SPOON、結局どっちがいい？
```

見出しのテンプレート：
- `〇〇って[疑問]？ [期間/実績]で検証した`
- `[サービス名]を[期間]使い続けてわかった[結論]`
- `正直に言う。[サービス名]の[向き不向き]`
- `[状況]な人に[サービス名]はどうか`

## 法則8：リズムと文体

**文章の長さのパターン（プロが使う）**

短 → 長 → 短 の繰り返し。

```
疲れた夜に、また考えなきゃいけないのか。

「今日、何食べる?」というLINEを見るたびに、
小さなため息をついていた時期がある。
共働き3年目、夫婦ともにフルタイム、子どもなし。
それでも毎晩のご飯問題は地味にしんどかった。

だからナッシュを使い始めた。
```

**文末のバリエーション（親近感を出す）**

単調になりがちな「〜です。〜ます。」を崩す：
```
続けてみた。 → 体言止め（インパクト）
悪くなかった。 → 過去形で淡々と
正直、びっくりした。 → 感情直球
思ったより好きかもしれない。 → 曖昧さが人間らしい
これは買いだと思う。 → 判断を示す
```

## 法則9：比較表の作り方

比較なしの「これがいい！」は信用されない。  
必ず競合と並べる。

```markdown
| 項目 | GREEN SPOON | ナッシュ | 三ツ星ファーム |
|-----|------------|---------|--------------|
| 1食あたり | 700〜900円 | 499〜764円 | 498〜690円 |
| 野菜量 | ★★★★★ | ★★☆ | ★★★ |
| 味の安定感 | ★★★★ | ★★★ | ★★★★ |
| こんな人向け | 健康意識高め | とにかく安く | バランス重視 |
```

表の直後に一言：
```
価格だけで選ぶならナッシュ一択。
でも「これで本当に栄養取れてるの?」という不安を消したいなら
GREEN SPOONかなと思う。
```

## 法則10：記事の最後の締め方

❌ 弱い締め（商品押し付け）
```
ぜひこの機会に申し込んでみてください！
```

✅ 強い締め（読者への問いかけ）
```
「時間を買う」という感覚に慣れるまで少し時間がかかった。
でも今は、平日の夜に「明日の弁当どうしよう」を考えない生活が
こんなに楽なのか、と思っている。

忙しさに疲れている誰かの参考になれば。
```

✅ 別パターン（条件付きおすすめ）
```
こんな人にはGREEN SPOONを試してほしい：
- 野菜不足が気になっている
- 「食事が面倒くさい」より「ちゃんとしたい」が少し勝っている
- 1食900円の投資を「安い」と感じられる

逆に、「とにかく安く」「量が多め」が優先なら、ナッシュの方が合う。
```

## 実際に使える「冒頭フレーズ集」

### 宅食・食事系
```
「今日も惣菜か…」と思いながらレジに並ぶ夜が、週に3回あった。
```
```
料理は嫌いじゃない。でも「何にするか考える」のが嫌いだ。
```
```
共働きを始めて最初に崩れたのは、ご飯じゃなくて「食生活への罪悪感」だった。
```
```
栄養バランスを気にする前に、まず「食べ続けられるか」が問題だと気づいた。
```

### 完全栄養食・機能性食品系
```
「これを食べれば栄養が取れる」は、面倒くさがりには神の言葉だった。
```
```
正直、最初は信じていなかった。「本当に1食でビタミン全部?」って。
3ヶ月後、体感は想像より良かった。
```

### ミールキット系
```
料理することは好きだった。ただ、平日夜22時に好きでいるのは限界だった。
```
```
「ちゃんと作った感」が欲しいのに、時間はない。
そのわがままを両立させてくれるものを探していた。
```

## まとめ：最重要ルール5つ

1. **問題 → 商品** の順番を絶対に守る（逆は売り込みになる）
2. **弱点を必ず書く**（書かないと信用されない）
3. **数字を具体的に**（「しばらく」より「3ヶ月・50食」）
4. **「こんな人に向いてる・向いてない」を書く**（全員向けは誰にも刺さらない）
5. **CTA（購入リンク）は信頼を作った後**（記事の後半か末尾）
