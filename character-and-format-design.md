# 時短×健康ブログ：キャラクター×掛け合い形式 設計書
> 作成: 2026-04-26
> 目的: 「理屈っぽい医師推薦」を「人間らしいドラマ形式」に転換する

---

## コンセプト一言

> **精神科医の「なぜ?」× 架空キャラの「で、実際どうだった?」**

医師の知識は信頼されるが感情移入しにくい。
架空キャラは親近感があるが信頼が薄い。
この2つを掛け合わせることで、**信頼×共感** を両立させる。

---

## 登場人物

### 【本物】長友恭平先生（精神科医・産業医）

- 実在する筆者本人
- 役割：「なぜ食事と健康がつながるか」を医学的・産業医的視点で解説
- 口調：丁寧だが断定的。「患者さんを診ていると〜」「研究では〜」
- 感情：あまり出さない。でも時々「これは正直おすすめできない」と辛口に

### 【架空】宮本ユウカ（34歳・架空のキャラクター）

| 項目 | 設定 |
|------|------|
| 年齢 | 34歳 |
| 職業 | IT系企業・プロジェクトマネージャー |
| 居住 | 東京・一人暮らし |
| 帰宅時間 | 平均21〜23時 |
| 食生活 | コンビニ飯が週4日。自炊は週末だけ |

**ユウカが代弁すること（＝読者の心の声）**
- 「宅食ってどうせ高い・まずい・続かないでしょ」という疑念
- 「でもコンビニ飯に罪悪感がある」という矛盾
- 「わかってるけどできない」という日常の摩擦
- 「先生の言ってることは正しいんだろうけど、現実は…」という抵抗感

## 記事フォーマット（掛け合いドラマ形式）

### 構成テンプレート

```
【状況設定】（ナレーション・2〜3行）
 → ユウカの今週の状況を第三者視点で描写

【掛け合いパート】
 [ユウカ] → 読者の心の声・愚痴・疑問・体験報告
 [先生]   → 医師視点の解説・背景・なぜそうなのか
 [ユウカ] → 感情的反応・「でも〜」という現実的抵抗
 [先生]   → 具体的なアドバイス・商品の文脈
 [ユウカ] → 実際に試した感想（良かった点・気になった点）
 [先生]   → 医学的な補足・「こういう人に合う・合わない」

【まとめパート】
 先生の総括（3〜5行）
 → 商品情報・リンク
```

### フォーマットのルール

1. **ユウカの発言は必ず「疑問・愚痴・抵抗」から始める**（いきなり「よかった！」にしない）
2. **先生は一度、ユウカに賛同してから反論する**（「確かにそれは正直なところで…でも〜」）
3. **商品名は掛け合いの中盤以降で出す**（設定＋問題提起の後）
4. **ユウカは必ず弱点・不満を1つ言う**（そこに先生が文脈を与える）
5. **締めはユウカが「まあ続けてみます」程度の温度感**（絶賛しない）

## サンプル記事：GREEN SPOON を試してみた話

> 記事ファイル名案: `greenspoon-review-drama.mdx`
> カテゴリ: review / drama

---

### [状況設定]

10月の第3週、宮本ユウカは連続4日コンビニ弁当を食べた。
悪いとわかっている。でも帰宅23時にスーパーは開いていない。

---

### [掛け合い]

**ユウカ：**
先生、また相談があるんですけど。
この前から「宅食使ってみれば?」って言ってますよね。
でも月に何万もかかるじゃないですか。それって現実的ですか?

**先生：**
気になって調べたんだよね。
コンビニ弁当を1食700〜800円として、週4日で1ヶ月だと約12,000円。
GREEN SPOONを週4で頼んだ場合、だいたい14,000〜16,000円くらい。
差は月2,000〜4,000円。ただし野菜量は全然違う。

**ユウカ：**
うーん、でも宅食って「なんか体に良さそうなフリしてる」だけで
実際大して変わらなくないですか?

**先生：**
それは正直、サービスによる。
私が気にするのは「野菜を主菜として設計しているか」という点。
GREEN SPOONはそこが比較的きちんとしていて、
「ハンバーグの横に少しの彩り野菜」じゃなくて、野菜がメインの弁当になっている。
精神科的に言うと、野菜不足は気力・集中力の低下と関連する研究があるから、
単に「体に良い」以上の意味がある。

**ユウカ：**
…なるほど。
じゃあ一応試してみようかな、と思って。
初回割引もあったので。

で、1週間使ってみた感想なんですけど——
正直に言うと、1食目はちょっと「これで足りる?」と思いました。
量が少なく感じて。コンビニのご飯に慣れてると、ボリュームが物足りない。

**先生：**
それは正直な感想で、合っていると思う。
GREEN SPOONは量より質のサービスだから、
「がっつり食べたい」人には向いていない。
ただユウカさんの悩みの本質は「量が足りない」じゃなくて「罪悪感と疲れ」でしょ。
そこはどうだった?

**ユウカ：**
……それは、変わりました。
「今日もちゃんとしたもの食べた」という感覚が残るというか。
コンビニ弁当を食べた夜とは違う気持ちで眠れる感じがして。
これ、気のせいですか?

**先生：**
気のせいじゃない。
「自分が自分を適切に扱えている」という感覚は、
精神医学的には自己効力感と関連していて、翌日のパフォーマンスにも影響する。
食事一つで大げさに聞こえるかもしれないけど、
毎日の「小さな罪悪感」の蓄積は、じわじわと気力を奪っていくんだよ。

**ユウカ：**
先生、今日はちょっと刺さりましたよそれ。
値段はやっぱり高めだと思うので毎日は無理ですけど、
週3日くらい続けてみます。

---

### [まとめ]

**先生より：**

GREEN SPOONは「健康意識があるけど行動できていない」人よりも、
「ちゃんとしたものを食べたいが時間がない」人に向いていると私は見ている。

産業医として企業を回っていると、
「コンビニ飯が続いている社員は精神的な疲労が蓄積しやすい」という傾向を感じる。
これは栄養の問題だけでなく、「自分への扱いが雑になっている」サインでもある。

全員に勧めるわけではないけれど、
ユウカさんのように「罪悪感がある」「でも変えられない」と感じているなら、
月数回の使用から試してみる価値はあると思う。

## 既存記事への適用方法

### パターンA：全面ドラマ形式（新記事）
新記事はすべて上記テンプレートで書く。

### パターンB：既存記事に「ユウカのひとこと」を追加（改修）
既存のレビュー記事の冒頭に、ユウカが疑問を投げかけるセクションを1ブロック追加するだけ。

例：`greenspoon-review-psychiatrist.mdx` の冒頭に追加
```
**ユウカ（34歳・IT系）より：**
> 先生のすすめるやつ、一回試してみたんですよ。
> 最初は「また理屈っぽいおすすめか」と思ってたんですが、
> 意外と…続いてます。以下に感想書いてもらいました。

---
（以降、既存の先生レビュー）
```

これだけでも「読者の代弁」と「冒頭の感情的な掴み」が生まれる。

### パターンC：DoctorComment コンポーネント と組み合わせ
既存の `DoctorComment` コンポーネントに `type="character-voice"` を追加し、
ユウカの発言を視覚的に区別する。

## 注意点（重要）

1. **先生の発言に「私が試した」とは書かない**
   → 先生は「医師の見解」「診療の観察」「研究」として語る。個人的な使用体験は書かない
   
2. **ユウカが「治った」「効果あった」と断言しない**
   → 「気持ちが楽になった」「罪悪感が減った」程度の温度感に留める

3. **先生が特定商品を「絶対おすすめ」と言わない**
   → 「こういう人に合う可能性がある」という条件付きの言い方に留める

4. **記事末尾の開示テキストを省略しない**
   → 「ユウカは架空」「アフィリリンクあり」の2点は全記事必須

5. **読者から「ユウカさんって本物ですか?」と聞かれた場合の答えを用意**
   → About ページで明示しているので「Aboutページをご覧ください」でOK
   → 「架空です」と積極的に言うことは誠実さの証明になるのでむしろ良い
