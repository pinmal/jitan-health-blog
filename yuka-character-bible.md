## 2. 性格・口調

### 性格の核
- **効率思考・めんどくさがり**：「どうせやるなら最短で」派
- **正直で自虐的なユーモア**：失敗談を隠さない。自分をオチにする
- **健康意識はあるが行動力が伴わない**：「わかってるけどできない」が口グセ
- **一度決めたら続ける**：始めるまでが重い。始めたら意外と続く

### 口グセ（記事全体で統一）
- 「でも高くないですか?」
- 「それ、続きます?」
- 「うーんどうだろ」
- 「まあ、いいかな（程度の温度感）」
- 「正直に言うと〜」

### 言わないこと・やらないこと（NG）
- ❌ 「最高でした！」「感動しました！」（絶賛しない）
- ❌ 「先生のおかげで人生変わりました」（大げさな感謝）
- ❌ 健康・食事への意識が急に高くなる（変化はゆっくり）
- ❌ 料理が得意になる（苦手なまま続く）
- ❌ 先生の意見にすぐ従う（「でも〜」「それでも〜」の抵抗が必ずある）
- ❌ 複数の新サービスを同時に試す（一つずつ慎重に試す人物）

## 6. 先生との関係・対話のルール

- ユウカとドクターは「ブログ上の対話」として描く（リアルの診察関係ではない）
- ユウカは先生のブログを読んでいる設定。コメント・質問を送る形
- **先生はユウカを患者として扱わない**（診察・治療の話にしない）
- ユウカは先生の意見に最初は「でも〜」と抵抗する。2〜3往復後にやや納得
- 最後はユウカが「まあ続けてみます」で締めるのが定型（完全同意しない）

## 10. キャラの「成長曲線」設計

ユウカは完璧にならない。でも少しずつ変わる。

```
開始時（2025年秋）
  └─ コンビニ飯 週4〜5日 / 朝食なし / 罪悪感MAX

3〜4ヶ月後（2026年初頭）
  └─ ナッシュで夕食改善 / BASE BREADで朝食確立 / 罪悪感が少し減る

現在（2026年4月）
  └─ 夕食パターン3つ確立 / 週末作り置き挑戦中 / 「まあ大丈夫」の状態

今後（半年〜1年）
  └─ 昼食改善を考え始める / 運動を検討する / ただし劇的な変化はしない
```

**重要**: ユウカは「理想の健康的な人」には絶対にならない。
「以前よりちょっとマシな状態が続いている」というリアリティが読者の共感を生む。

## 11. 文体見本（正解と不正解）

### 正解
```
「帰りが22時超えたときは、もうナッシュ一択です。
 考えるのが面倒くさいし、考えたくもない。」

「BASE BREADが2ヶ月続いているのは、正直自分でも驚いてます。
 ただ、おいしいから続いてるんじゃなくて、考えなくていいから続いてる。」

「作り置き、また今週もできませんでした。
 …まあ、ナッシュあるし、いいか。」
```

### 不正解
```
❌「先生のアドバイスのおかげで食生活が整いました！」
❌「GREEN SPOONはとても美味しくて感動しました！」
❌「健康を意識するようになってから毎朝野菜ジュースも飲んでいます」
❌「料理が楽しくなってきました」
```
