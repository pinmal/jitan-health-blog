## 「何をつくるか」を決めることが、一番疲れる

夕方、仕事が終わってから「今日の夕食、何にしよう」と考える。

この一言に、想像以上のエネルギーが使われています。

メニューを考え、食材を確認し、足りないものをリストアップして買いに行き、料理する。この流れをゼロから組み立てるコストは、特に疲れているときには想像以上に大きい。

<DoctorComment type="experience">
外来で「最近、食事が乱れています」という話になると、多くの場合「何を食べるか決められなくて、結局コンビニになる」という流れが見えてきます。料理が嫌いなわけでも、食欲がないわけでもなく、「決める」ところで力が尽きてしまっている。
</DoctorComment>

ミールキットは、この「決める」部分をほぼゼロにする仕組みです。

## ヨシケイとはどんなサービスか

ヨシケイは、**食材とレシピがセットで届く定期配送型のミールキットサービス**です。

届いた食材をレシピ通りに調理するだけで夕食が完成します。食材の調達・献立決定という2大コストを同時に削除できます。

主な特徴：

| 項目 | 内容 |
|---|---|
| 提供形式 | 食材＋レシピのセット配送 |
| 週単位 | 週ごとにメニューが変わる |
| 配達エリア | 全国（一部地域を除く） |
| 食数 | 2人前・3人前など選択可能 |
| 調理時間 | 目安15〜20分程度 |

ポイントは**「毎週メニューが変わる」**点です。「いつもと同じ献立」に飽きる問題を解決しながら、献立を自分で考える必要がありません。

## 精神科医として評価する理由

### 食事の「決断コスト」を構造的に下げる

心理学では「選択のパラドックス」という概念があります。選択肢が多すぎると、かえって意思決定が困難になり、満足度も下がる、という現象です。

ミールキットは「今日はこれを作る」と決まっているため、選択肢がゼロです。**判断不要＝疲れない**という構造になっています。

| 食事の準備スタイル | 決断回数 |
|---|---|
| 献立をゼロから考える | 多（何を作るか・何を買うか） |
| 冷蔵庫の余り物で考える | 中（何が使えるか判断が必要） |
| ミールキットを作る | 低（レシピ通りに動くだけ） |
| 宅配弁当を温める | ほぼゼロ |

<DoctorComment type="opinion">
ミールキットと宅配弁当はよく比較されますが、目的が少し違います。宅配弁当は「料理しない日」に向いていて、ミールキットは「料理はしたいが献立は考えたくない日」に向いている。どちらが良いかではなく、どちらもツールとして使い分けるのが現実的だと思います。
</DoctorComment>

### 栄養バランスが設計されている

ヨシケイのミールキットは管理栄養士が栄養バランスを考慮して設計されています。

「何を食べるか考えられない日」は、栄養バランスも崩れやすい日でもあります。こういった日に、バランスが設計された食事が自動的に届く仕組みは、健康管理の観点から見ても合理的です。

### 食材の無駄を出さない

食材を自分で購入すると、使いきれずに余る問題が起きます。特に一人暮らしや少人数世帯では、食材の廃棄が日常的なストレスになることがあります。

ミールキットは必要な分量の食材がセットになっているため、**余り食材の管理という判断コストも消える**というメリットがあります。

## 向いている使い方・向いていない使い方

**向いていると感じた使い方：**
- 週の固定曜日（例：月・水）にミールキットの日を設定する
- 「料理はしたいが献立は考えたくない」という中間的なニーズに対応
- 食費の見通しを立てやすい（定額で月の食費が計算しやすい）

**向いていないと感じた場面：**
- 急な残業で帰宅が遅くなった日（調理時間が取れないとき）
- 完全に疲れきっていて何もしたくない日（宅配弁当のほうが適切）

<DoctorComment type="caution">
ミールキットは「調理工程が残る」点を忘れないでください。疲れてどうしても動けない日のために、電子レンジのみで完結する冷凍弁当もセットでストックしておくことを推奨します。ミールキットとバックアップ食材の2層構造が、食事の安定につながります。
</DoctorComment>

## 仕組み化の中でのヨシケイの位置づけ

週間食事テンプレートの例：

| 曜日 | 食事の方針 |
|---|---|
| 月曜日 | ヨシケイのミールキット |
| 火曜日 | 宅配弁当（DELIPICKS等） |
| 水曜日 | ヨシケイのミールキット |
| 木曜日 | 宅配弁当 or バックアップ食材 |
| 金曜日 | 外食・テイクアウト自由 |
| 土日 | 時間があれば自炊、なければ弁当 |

この構造をつくると、「今日は何にしよう」という問いが、「今日はミールキットの日」に変わります。
